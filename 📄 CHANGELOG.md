# Changelog

All notable changes to the Bangladesh ISP MikroTik Ultimate Setup will be documented in this file.

## [5.0.0] - 2024

### Added
- Complete IPv6 support with SLAAC/DHCPv6/6to4
- IPsec VPN with IKEv1/IKEv2 support
- OpenRoaming / Hotspot 2.0 support
- RadSec (Secure RADIUS) support
- Container (Docker) support
- MPLS/VPLS support
- Layer 7 filtering
- Telegram bot alerts
- SNMPv3 monitoring
- Interface bonding (LACP)
- Zero-touch provisioning
- Disaster recovery scripts

### Changed
- Upgraded to RouterOS 7.x compatibility
- Enhanced DDoS protection
- Improved QoS with CAKE/FQ_CODEL
- Better failover mechanism
- Optimized firewall rules
- Enhanced security hardening

### Fixed
- All known bugs from previous versions
- Memory leak issues
- Performance optimization
- Script error handling

## [4.0.0] - 2023

### Added
- BGP/OSPF routing protocols
- VRRP high availability
- WireGuard VPN
- Web proxy with caching
- Auto backup system
- Email alerts

### Changed
- Modular configuration structure
- Better error handling
- Improved documentation

## [3.0.0] - 2022

### Added
- PPPoE with RADIUS
- Hotspot with walled garden
- Basic VPN services
- Firewall with DDoS protection
- QoS bandwidth management
- BDIX routing

### Changed
- Initial production release
- Complete Bangla documentation
